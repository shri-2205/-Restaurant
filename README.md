# 🍽️ Homie Restaurant

> A modern, elegant fine-dining restaurant website built with pure HTML, CSS & JavaScript.

---

## 🌐 Live Demo

**👉 [https://shri-2205.github.io/-Restaurant/](https://shri-2205.github.io/-Restaurant/)**

---

## ✨ Features

- **Custom Cursor** — Gold dot + ring cursor for a luxury feel
- **Smooth Hero Animation** — Slow zoom-in with fade-up text
- **Tabbed Menu System** — Starters, Pasta, Burgers, Pizza, Salads, Desserts
- **Scroll Reveal Animations** — Elements fade in as you scroll
- **Interactive Gallery Strip** — Hover to expand photos
- **Reservation Form** — Full booking form with date, time & guest picker
- **Sticky Navbar** — Transparent → frosted glass on scroll
- **Fully Responsive** — Works on desktop and tablet

---

## 🛠️ Tech Stack

| Technology | Usage |
|-----------|-------|
| HTML5 | Structure & semantic markup |
| CSS3 | Styling, animations, grid & flexbox |
| Vanilla JavaScript | Cursor, tabs, scroll reveal, navbar |
